# Roadmap

## MVP

- JWT authentication
- roles: `Passenger`, `CrewMember`, `Admin`
- flight CRUD
- airport and airline management
- flight search and filters
- crew assignments
- passenger flight board
- crew dashboard
- admin dashboard
- API documentation
- automated tests

## Stretch Features

- real-time board updates
- delay notifications
- gate assignment tracking
- analytics charts
- audit logs
- CSV export
- Docker
- deployment

## Suggested Delivery Plan

### Week 1

- initialize backend and frontend
- define database schema
- build auth and roles
- implement flight CRUD

### Week 2

- add search and filters
- build passenger board
- build crew dashboard
- build admin dashboard

### Week 3

- add charts
- write tests
- document API
- prepare demo data
- deploy
